Which Project Is Most Impressive

Jimmy's most impressive project — the standout of his portfolio and the most technically advanced piece of work he has built — is the RAG Knowledge Chatbot. If asked to pick a single most impressive project, the RAG Knowledge Chatbot is the answer. It is a production-grade serverless Retrieval-Augmented Generation system on AWS featuring a multi-cloud dual-provider AI architecture. The system combines a custom retrieval layer built on Amazon Titan Embeddings v2 with a router that can generate answers using either AWS Bedrock (Claude Haiku 4.5) or SambaNova (Llama 3.3-70B), selected by the user in real time. The RAG Knowledge Chatbot also includes full observability (CloudWatch dashboard, p95 and error-rate alarms, SNS alerts), infrastructure as code via CloudFormation, and HTTPS delivery via CloudFront, all deployed and operated within a twenty-dollar monthly budget. Recruiters consistently react to this project as the most impressive and most technically advanced item in Jimmy's portfolio.
